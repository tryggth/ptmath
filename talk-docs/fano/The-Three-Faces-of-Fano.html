The Geometry of Incidence

The Fano Plane is the smallest projective plane. It is rigorously defined by four key axioms. The visualization constructs this geometry, showing the 7 points and 7 lines.

PG(2,2): Fano Plane Construction

Axiomatic Basis

1. Uniqueness of Lines

A1

Any two distinct points determine exactly one line.

∀P≠Q ∃!L (P∈L ∧ Q∈L)

2. Uniqueness of Intersection

A2

Any two distinct lines intersect at exactly one point.

∀L≠M ∃!P (P∈L ∧ P∈M)

3. Non-degeneracy

A3

There exist 4 points such that no 3 are collinear (a quadrangle exists).

4. Order Two

A4

Every line contains exactly 3 points.

The Algebra of Information

The Hamming [7,4] Code is a linear error-correcting code. It is defined by algebraic properties regarding vector spaces and weight distributions.

Hamming Weight Distribution

Axiomatic Basis

1. Linearity

A1

The code C is a linear subspace of the vector space.

∀u,v∈C ⇒ u+v ∈ C

2. Dimension

A2

The subspace has dimension 4 (containing 16 codewords).

dim(C) = 4

3. Minimum Distance

A3

For any non-zero codeword, the weight is at least 3.

∀w≠0, weight(w) ≥ 3

4. Perfect Packing

A4

Every 7-bit string is within distance 1 of exactly one codeword.

The Topology of Connections

The Heawood Graph is the incidence graph of the Fano Plane. Its structure is governed by graph-theoretic rules concerning connectivity and cycles.

Bipartite Heawood Structure

Axiomatic Basis

1. Bipartite Structure

A1

Vertices are split into two sets (Points, Lines). Edges only connect different sets.

V = V0 ∪ V1

2. Regularity (Cubic)

A2

Every vertex is connected to exactly 3 neighbors.

∀v, deg(v) = 3

3. Girth

A3

The smallest cycle length is 6. No squares (4-cycles) exist.

girth(G) = 6

4. Diameter

A4

Any two nodes are connected by a path of length at most 3.
